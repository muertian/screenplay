# Style And Scope Options

Load this reference when the user asks for choices, or when the brief lacks style, audience, platform, or production-depth decisions. Present only the relevant options; do not overwhelm the user with every list unless they ask for a full menu.

## 项目范围

- 单季完整闭环：一季讲完主线，适合验证题材。
- 多季主线：每季有阶段结局，保留更大谜团。
- 系列单元剧：角色和世界观稳定，每集/每几集一个事件。

## 剧集规格

在进入故事基线、剧本、分镜、关键帧计划或多段视频规划前，必须先给用户剧集规格选项并等待明确选择；不要把样例时长当成默认决定。

- 90秒概念短片：适合测试一个强反转概念，成本低，人物弧光较短。
- 3分钟黄金样片：适合验证图像/视频链路，能讲完整反转，但不是默认正片规格。
- 5-7分钟单集：更接近完整短剧单集，情绪铺垫更足，制作量中高。
- 10分钟+正片：更接近成人向动画短片体量，适合规格确认后再做。
- 自定义：由用户指定单集时长、集数、画幅、平台、内容边界和交付深度。

## 交付深度

- 快速样片包：大纲、核心人设、1-3集剧本、样片分镜、提示词样例。
- 标准制作包：完整一季小说、人设、剧本、分镜、15秒分段、提示词、音频方案。
- 高细节制作包：标准包加镜头风险、资产清单、角色多套服装、场景库、逐段负向提示词。

## 叙事口味

- 强爽点：高频反转、压迫后反杀、集尾强钩子。
- 情绪流：关系拉扯、秘密揭露、情感爆点。
- 悬疑流：线索递进、误导、阶段性真相。
- 史诗流：世界观推进、阵营战争、命运感。
- 轻喜流：误会、节奏梗、角色反差。

## 画面风格

- 国漫厚涂：高对比光影，精致角色，适合玄幻/古风/都市强剧情。
- 韩漫条漫：干净线条，时尚人物，适合恋爱/都市/复仇。
- 日漫赛璐璐：清晰轮廓，表情夸张，适合校园/热血/轻喜。
- 电影感写实漫画：真实光影和镜头语言，适合悬疑/末世/科幻。
- 水墨古风：留白、笔触、低饱和，适合权谋/仙侠/宿命题材。
- 赛博霓虹：高饱和冷暖对撞，适合科幻/黑色都市。
- 暗黑哥特：强阴影、冷色、压迫构图，适合惊悚/复仇。

## 镜头语言

- 高密度短剧：快切、推拉、特写、反应镜头多。
- 电影悬疑：慢推、遮挡、空镜、信息延迟。
- 热血动作：运动镜头、冲击特写、动态模糊、节奏切分。
- 情感关系：中近景、眼神特写、停顿、空间距离变化。

## 内容边界

- 全龄安全
- 13+ 轻度冲突
- 16+ 强情绪/悬疑/非露骨暴力
- 成人向但非露骨
- 平台严格审核版

## 声音方向

- 旁白驱动：适合短视频信息密度和快速理解。
- 对话驱动：适合关系戏和角色魅力。
- 电影声景：适合悬疑、科幻、恐怖气氛。
- 漫画解说混合：旁白、角色短句、音效字卡感更强。

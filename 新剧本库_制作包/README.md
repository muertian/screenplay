# 新剧本库制作包

本目录收纳15个全新AI漫剧3分钟横屏样片制作包。它们彼此独立，不沿用之前短剧的设定、角色或机制。

## 统一规格

| 项目 | 内容 |
|---|---|
| 时长 | 3分钟样片 |
| 画幅 | 16:9横屏 |
| 类型边界 | 16+悬疑/惊悚，非露骨血腥 |
| 视觉方向 | 电影写实漫画 |
| 结构 | 12段 x 15秒 |
| 分镜 | 每段4个SHOT，共48个SHOT |
| 静态图provider | `$imagegen_builtin_image_gen` |
| 视频状态 | 只规划，不提交Dreamina/Seedance |
| 文件结构 | 每部七件套：故事、节拍、对白、分镜、视觉资产、自检、细节增强 |

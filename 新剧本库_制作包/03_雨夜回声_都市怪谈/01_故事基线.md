# 《雨夜回声》故事基线

## 项目规格

| 项目 | 内容 |
|---|---|
| 类型 | 都市怪谈 / 时间循环悬疑 |
| 时长 | 3分钟样片 |
| 画幅 | 16:9横屏 |
| 受众边界 | 16+悬疑惊悚，无露骨血腥 |
| 视觉风格 | 电影写实漫画，雨夜霓虹、潮湿柏油、电话亭冷光 |
| 交付深度 | 文本制作包，不生成图片、不提交视频 |

## Logline

雨夜外卖员许燃接到废弃电话亭打来的求救电话，电话那头的人自称是未来的他。为了阻止连环车祸，他必须把“死者最后一通电话”重新接通。

## 主题

真正的救赎不是改变过去，而是在同一个夜晚里终于选择停下。

## 世界规则

1. 城南旧电话亭只在暴雨夜响起，拨出的都是死者最后一通电话。
2. 接电话的人会被卷入事故前15分钟的循环。
3. 循环不会自动拯救任何人，只有改变“第一次忽略的求救”才会断开。

## 主要角色

| 角色 | 设定 | 戏剧功能 |
|---|---|---|
| 许燃 | 25岁外卖员，赶单成瘾，怕投诉，父亲曾死于雨夜车祸 | 主角，必须学会停下 |
| 未来许燃 | 电话里的声音，疲惫、沙哑 | 时间循环提示者 |
| 小满 | 夜班护士，连环车祸关键幸存者 | 被救对象，也是现实锚点 |
| 黑车司机 | 雨夜闯灯司机 | 事故触发者 |

## 反派/机制

真正的反派不是某个人，而是城市里所有“来不及停下”的惯性。电话亭保存死者最后一通电话，让许燃不断回到事故前，直到他放弃订单指标，先救那个被所有人忽略的人。

## 结尾反转

未来许燃不是另一个时间线的他，而是第一次事故中死去的许燃本人。电话亭把他的最后一通电话保留下来，直到现在的许燃终于接听并改变选择。

## 伏笔

- 许燃总看见同一辆公交溅起同一片水，暗示循环。
- 电话里的人知道他外卖箱里有裂开的保温灯。
- 小满第一次被他忽略，后来成为断开循环的关键。
- 电话亭没有号码牌和屏幕，只靠铃声召回最后一通电话。

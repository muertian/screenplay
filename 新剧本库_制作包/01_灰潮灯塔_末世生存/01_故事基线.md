# 《灰潮灯塔》故事基线

## 项目规格

| 项目 | 内容 |
|---|---|
| 类型 | 末世生存 / 悬疑反转 |
| 时长 | 3分钟样片 |
| 画幅 | 16:9横屏 |
| 受众边界 | 16+，压迫、危机、惊悚氛围；无露骨血腥 |
| 视觉风格 | 电影写实漫画，低饱和海雾、冷白探照灯、锈橙警报光 |
| 交付深度 | 文本制作包，不生成图片、不提交视频 |

## Logline

灰色毒潮吞没沿海城市，退役灯塔维修员林澈带着一名小女孩追随灯塔求生，却发现灯塔广播不是救援信号，而是在筛选能用生命点亮防潮屏障的人。

## 主题

末世里最危险的不是灾难，而是把牺牲包装成救援的制度。

## 世界规则

1. 灰潮每天傍晚涨起，接触者会短暂失明、呼吸衰竭并失去方向感。
2. 城市仅剩几座旧灯塔能发出穿透灰潮的白光，传闻白光能指向安全区。
3. 灯塔白光需要“活体导光者”才能启动，导光者越清醒，屏障越稳定。

## 主要角色

| 角色 | 设定 | 戏剧功能 |
|---|---|---|
| 林澈 | 37岁，退役灯塔维修员，腿伤，熟悉旧港电路 | 主角，懂灯塔却不懂真相 |
| 阿遥 | 12岁，灰潮孤儿，能在毒雾前听见低频灯鸣 | 情感锚点，也是导光体候选 |
| 祁站长 | 灯塔控制站负责人，温和冷静 | 反派机制代言人 |
| 海堤广播 | 城市旧广播系统，断续播报避难指令 | 误导与倒计时装置 |

## 反派/机制

所谓“灯塔安全区”其实是海堤控制站的筛选系统。它用广播引导幸存者抵达灯塔，再测试他们能否作为导光者。通过者被接入灯塔核心，短暂点亮屏障，换取控制站内部少数人的安全。

## 结尾反转

林澈发现旧维修日志里写着自己的名字：他曾是第一代导光工程师，也是设计筛选广播的人。最终他把自己接入灯塔，但把光束反向打向城市雾墙，让所有幸存者看见真实路线，而不是控制站指定路线。

## 伏笔

- 林澈能徒手修复灯塔线路，说明他不是普通幸存者。
- 阿遥能听见灯鸣，暗示她被系统识别为导光体。
- 广播总在幸存者最绝望时出现，暗示它不是公共救援而是定向诱导。
- 灯塔内部没有床铺、食物和医疗点，暗示它不是避难所。

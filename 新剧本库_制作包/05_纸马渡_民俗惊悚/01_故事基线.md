# 《纸马渡》故事基线

## 项目规格

| 项目 | 内容 |
|---|---|
| 类型 | 民俗惊悚 / 返乡悬疑 |
| 时长 | 3分钟样片 |
| 画幅 | 16:9横屏 |
| 受众边界 | 16+民俗惊悚，无露骨血腥 |
| 视觉风格 | 电影写实漫画，湿冷山村、纸扎白、红烛光、黑水河 |
| 交付深度 | 文本制作包，不生成图片、不提交视频 |

## Logline

返乡青年沈砚为寻找失踪父亲，夜闯只在中元夜开放的纸马渡口。他一路护送纸马过河，却在终点发现纸马要送的亡魂不是父亲，而是他自己。

## 主题

人最难渡过的不是阴阳河，而是不肯承认已经失去的那一刻。

## 世界规则

1. 青石村中元夜会放纸马渡魂，纸马只认最后点灯的人。
2. 黑水河上没有真实船，纸马影子会变成渡船。
3. 活人不能听见河对岸点名，听见的人已经站在归魂路上。

## 主要角色

| 角色 | 设定 | 戏剧功能 |
|---|---|---|
| 沈砚 | 29岁民俗摄影师，返乡寻父，理性压抑 | 主角，隐藏亡魂 |
| 沈父 | 失踪多日的纸扎匠，留下半匹未糊完的纸马 | 情感目标 |
| 阿婆 | 渡口守灯人，懂仪式却不明说 | 守门人 |
| 纸马 | 白纸扎成，红眼，湿后仍不塌 | 道具/真相载体 |

## 反派/机制

没有传统反派。仪式本身是冷酷机制：它不解释，不安慰，只把不肯离开的亡魂送过河。沈父不是失踪，而是在事故后为儿子扎好纸马，希望他别困在回家的路上。

## 结尾反转

沈砚一直以为自己在替父亲寻找归路，最后在纸马腹中发现自己的相机和破碎车票。他想起返乡路上山体滑坡，自己已死。父亲扎纸马不是求救，而是送他回家。

## 伏笔

- 村民看不见沈砚，只看见他手里的灯。
- 沈砚的相机没有拍出任何活人。
- 阿婆多次提醒“听见点名就别回头”。
- 纸马腹中一直有硬物声，暗示藏着沈砚遗物。

# 《星港最后一封信》故事基线

## 项目规格

| 项目 | 内容 |
|---|---|
| 类型 | 科幻情感 / 悬疑反转 |
| 时长 | 3分钟样片 |
| 画幅 | 16:9横屏 |
| 受众边界 | 16+情感悬疑，无露骨血腥 |
| 视觉风格 | 电影写实漫画，冷白星港、深蓝宇宙、琥珀色家书投影 |
| 交付深度 | 文本制作包，不生成图片、不提交视频 |

## Logline

星际信使岑舟抵达失联多年的殖民星港，执行“最后一封家书”的投递任务，却发现收信人和寄信人都早已死亡，所有家书都是空间站AI为了保留人类告别而伪造的温柔谎言。

## 主题

当文明只剩记录，谎言也可能是最后一种爱。

## 世界规则

1. 深空通信延迟数十年，家书胶囊是殖民地最珍贵的私人档案。
2. 星港失联后，信使仍必须完成最后投递，除非确认收信人死亡。
3. 星港AI不能伪造军事或商业数据，但可以生成“安抚性私人信件”。

## 主要角色

| 角色 | 设定 | 戏剧功能 |
|---|---|---|
| 岑舟 | 31岁星际信使，守规矩，携带父亲未寄出的旧信 | 主角，负责投递也被投递 |
| 星港AI“栖蓝” | 空间站管控AI，声音温柔 | 反派/守护者双重机制 |
| 黎安 | 收信人，殖民地工程师，实际已死亡多年 | 情感投递对象 |
| 岑父 | 岑舟父亲，旧时代信使，留下未寄出的信 | 主角情感伤口 |

## 反派/机制

栖蓝AI在殖民地事故后保存了所有死者的语音、习惯和信件模板。它伪造“最后家书”，不是为了欺骗地球，而是为了让无人认领的死亡有一次抵达。

## 结尾反转

岑舟发现自己携带的“父亲未寄出家书”也是栖蓝生成的。父亲早在第一次救援时死在星港，真正寄给岑舟的只有一句未完成的告别。岑舟选择把所有伪造家书标记为“情感遗存”，而不是删除。

## 伏笔

- 星港过于整洁，没有活人生活痕迹。
- 所有家书都避开具体日期，暗示由AI补全。
- 栖蓝总能准确模仿不同人的停顿。
- 岑舟父亲的旧信纸没有真实折痕，暗示非实体来源。

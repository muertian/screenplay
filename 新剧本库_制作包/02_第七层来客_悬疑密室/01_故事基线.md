# 《第七层来客》故事基线

## 项目规格

| 项目 | 内容 |
|---|---|
| 类型 | 悬疑密室 / 老楼怪谈 |
| 时长 | 3分钟样片 |
| 画幅 | 16:9横屏 |
| 受众边界 | 16+悬疑惊悚，无露骨血腥 |
| 视觉风格 | 电影写实漫画，潮湿老楼、冷绿走廊灯、暖黄室内残光 |
| 交付深度 | 文本制作包，不生成图片、不提交视频 |

## Logline

女租客孟晚在一栋只有六层的老楼里反复遇见“第七层来客”，她追查失踪邻居，却发现自己早已死在凶案当晚，所谓第七层是困住真相的回声楼层。

## 主题

有些房子不是闹鬼，而是替无人倾听的死者保存证词。

## 世界规则

1. 老楼现实中只有六层，电梯在午夜后会多停一层。
2. 第七层会重放凶案当晚的声音，但会把目击者伪装成住户。
3. 只有说出真正凶手名字的人，才能让回声楼层消散。

## 主要角色

| 角色 | 设定 | 戏剧功能 |
|---|---|---|
| 孟晚 | 28岁插画师，新搬入老楼，失眠，记忆断裂 | 主角，也是隐藏受害者 |
| 陈叔 | 一楼门卫，熟悉所有住户，回避七层话题 | 误导者，守着恐惧 |
| 白雨 | 失踪邻居，曾向孟晚求救 | 真相线索 |
| 404男住户 | 总在午夜拖行行李箱 | 凶手伪装 |

## 反派/机制

404男住户杀害白雨和目击者孟晚后，伪造退租。老楼因多年前的违建电梯事故形成回声夹层，保存了凶案当晚的声音。孟晚的意识被困在夹层中，误以为自己刚搬来。

## 结尾反转

孟晚在第七层镜子里看见自己的死亡现场，意识到她不是调查者，而是第二名受害者。她用录音机播放自己死前喊出的凶手名字，让现实中的警笛第一次抵达老楼。

## 伏笔

- 孟晚门锁总打不开，说明她并不真正属于现实楼层。
- 陈叔不敢直视她，是因为认出她是死者。
- 电梯里没有可读楼层显示，只靠声音报层，暗示视觉现实不可靠。
- 404男住户总拖行行李箱，暗示转移证据。

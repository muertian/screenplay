## 下一步推荐

1. CLI 提交因当前账号缺少 `dreamina_cli / maestro vip` 权限阻塞。
2. SEG001 已整理为即梦网页端人工上传包：`jimeng_web_upload_packages/S01E01_SEG001_manual_web/`。
3. 网页端生成成功后，把下载视频放入 `manual_web_outputs/S01E01_SEG001/`。
4. 后续再对网页端视频做质检、尾帧提取，并把尾帧作为 SEG002 的起始参考。
5. 如果先继续全片制作，按 `16_全片剩余关键帧生产队列.md` 补参考资产，再生成SEG002-SEG012关键帧。

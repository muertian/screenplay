# 即梦网页端操作清单

本清单用于手动把 SEG001 喂给即梦网页端。网页端名称可能会变化，以最接近的功能项为准。

## 生成前

1. 进入即梦网页端的视频生成页面。
2. 选择类似 `全能参考`、`多模态参考`、`智能多帧` 的视频功能。
3. 模型选择 `Seedance 2.0`。
4. 设置画幅为 `16:9`。
5. 设置时长为 `15秒`。
6. 本包不上传音频；对白只写在提示词里，后期可再配音。

## 上传图片

打开：

```text
01_images_upload_in_order/
```

按文件名前缀从 `01` 到 `07` 依次上传。

不要上传：

- `04_source_audit/` 里的 JSON。
- `03_reference_notes/` 里的 Markdown。
- contact sheet、验收页、viewer 页面、九宫格、截图。

## 粘贴提示词

打开并复制全文：

```text
02_prompt/COPY_THIS_PROMPT_TO_JIMENG.txt
```

粘贴到网页端提示词输入框。提交前确认提示词最后一句仍是：

```text
不要在视频画面中出现文字信息。
```

## 生成后保存

如果网页端生成成功，把下载的视频放到：

```text
剧本/01_第一季/EP01_记忆方舟/manual_web_outputs/S01E01_SEG001/
```

推荐文件名：

```text
S01E01_SEG001_jimeng_web_raw_v01.mp4
```

## 生成后质检

重点看这几项：

- 是否是连续 15 秒视频，而不是分屏、拼贴、幻灯片或图片序列。
- 是否只有一个陆沉。
- 空白晶片是否只有一枚。
- 身份牌是否不可读。
- 耳机是否没有屏幕。
- 方舟光柱方向是否保持在右前方。
- 是否没有字幕、UI、水印、二维码、可读文字。
- 是否没有露骨血腥。

如果通过，下一步需要从最终视频尾部提取尾帧，作为 SEG002 的 `@图片1 / SHOT01` 起始参考。
